# Installation

Follow these steps to install MAAIS-Runtime:

## Option 1: Install from PyPI

```bash
pip install maais-runtime
```

## Option 2: From source

```bash
git clone https://github.com/MasterCaleb254/maais-runtime.git
cd maais-runtime
pip install -e .
```

## Optional extras
```bash
pip install maais-runtime[ml,webhooks,enterprise]
```
